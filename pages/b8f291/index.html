# 开始之前

请确保可以正常打开和使用 GitHub (opens new window)

请先对以下内容有一定了解:

  • vuepress (opens new window)
  • vuepress-theme-vdoing (opens new window)
  • git (opens new window)

# 安装软件

安装以下软件：

  • Node.js LTS (opens new window): 选择长期维护版，请勿使用最新尝鲜版，国内用户可选择 npmmirror 中国镜像站 (opens new window)下载
  • git (opens new window)
  • Visual Studio Code (opens new window): 编辑器，可选
  • TortoiseGit (opens new window): Git 可视化操作工具，可选
